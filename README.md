# Amazon Review Sentiment Analysis

A Data Science and Natural Language Processing project that analyzes Amazon product reviews and predicts customer sentiment using machine learning techniques. The system processes large-scale review datasets, performs text preprocessing, extracts meaningful insights, and classifies reviews into positive or negative sentiment categories.

---

## Features

- Review text preprocessing and cleaning
- Sentiment classification using Machine Learning
- Exploratory Data Analysis (EDA)
- Data visualization for customer insights
- NLP pipeline for text tokenization and feature extraction
- Performance evaluation using classification metrics

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

The project uses Amazon product review datasets containing:
- Review text
- Ratings
- Product information
- Customer feedback metadata

---

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Text Vectorization using TF-IDF
4. Model Training and Evaluation
5. Sentiment Prediction
6. Visualization and Insights Generation

---

## Models Used

- Logistic Regression
- Naive Bayes
- Random Forest
- Support Vector Machine (SVM)

---

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/your-username/amazon-review-sentiment-analysis.git
```

Move into the project directory:

```bash
cd amazon-review-sentiment-analysis
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
